**NOME.ART** es una interfaz de portafolio minimalista diseñada para artistas y músicos modernos. El proyecto se centra en una experiencia de usuario inmersiva utilizando técnicas avanzadas de CSS como Glassmorphism, animaciones de entrada y texturas granulares.

## 💎 Características Principales

- **Glassmorphism UI:** Implementación de `backdrop-filter` para un efecto de cristal esmerilado premium.
- **Micro-interacciones:** Animaciones fluidas en hovers, carga de página y estados de reproducción.
- **Ambient Lighting:** Sistema de iluminación radial dinámico que reacciona visualmente.
- **Responsive Design:** Optimizado para dispositivos móviles y escritorio.
- **Arquitectura de Código Senior:** Uso intensivo de variables CSS para facilitar el re-branding instantáneo.

## 🛠️ Tech Stack

- **Estructura:** HTML5 Semántico.
- **Estilos:** CSS3 Moderno (Custom Properties, Grid, Flexbox).
- **Tipografía:** Syncopate & Inter (Google Fonts).
- **Assets:** Imágenes optimizadas vía Unsplash API.

## 🎨 Personalización

El proyecto está construido sobre un sistema de variables globales. Para cambiar el color de acento o el fondo, solo necesitas editar el `:root` en `style.css`:

```css
    :root {
        --accent: #ff003c; /* Cambia este valor */
        --bg-dark: #050505;
    }
